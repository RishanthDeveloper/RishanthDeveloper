## 🚀 About Me

I hold **ServiceNow CSA and CSD certifications** and enjoy working across the stack — from full-stack web apps to blockchain-based systems and AI-assisted tools.

I like taking a project from idea → deployable product, not just a prototype — with clean documentation and GitHub-ready packaging every time.
